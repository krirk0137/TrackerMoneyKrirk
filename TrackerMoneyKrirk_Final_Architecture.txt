Discord Notes
=============

Priority #1

Pros

- ฟรี
- API ดีมาก
- ส่งรูปง่าย
- Dev ง่าย
- เหมาะกับโปรเจกต์ส่วนตัว

สามารถ Deploy Bot

- Render
- Railway
- VPS
- PC ที่บ้าน


Telegram Notes
==============

Priority #2

Pros

- ฟรี
- Bot API ดีมาก
- ทำง่ายใกล้เคียง Discord

สามารถใช้ Backend เดียวกันกับ Discord


LINE Notes
==========

Priority #3

Pros

- คนไทยใช้เยอะ
- Webhook ฟรี

Cons

- Setup ยุ่งยากกว่า
- มีข้อจำกัดมากกว่า Discord/Telegram

สามารถใช้ Backend เดียวกัน


Supabase Notes
==============

ควรมี Edge Function

ชื่อ

process-message

หน้าที่

1. รับข้อความ
2. เรียก Qwen
3. Parse JSON
4. Save Database

ควรมี Edge Function

process-image

หน้าที่

1. รับ URL รูป
2. เรียก Qwen OCR
3. Save Database
